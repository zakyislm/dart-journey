Dart dan Perkembangannya

Memahami Fundamental bahasa pemrograman Dart, Tujuan, serta Penggunaannya

calendar_today 06-05-2026
Dart
Dart adalah bahasa pemrograman yang dibuat google pada 2012 dengan tujuan menciptakan alternatif modern untuk menggantikan Javascript sebagai bahasa pemrograman untuk aplikasi web. Google sebagai pencipta terus mengembangkannya, dan menjadi populer berkat perilisan framework UI Flutter, dimana Dart menjadi bahasa pemrograman utama framework ini.
Dalam pengembangannya, Dart membuat solusi yang sangat membantu developer, seperti: 1. Null Safety: mencegah null pointer exception \n2. Modernized Compiler: fitur hot reload & just-in-time \n3. OOP: semua value di Dart adalah Object
Sekarang, environment Dart telah berkembang sangat pesat yang melebihi tujuan penciptaannya, seperti: 1. Pembuatan Software cross platform Dart digunakan sebagai bahasa pemrograman di Flutter untuk membuat aplikasi yang dapat dijalankan dari berbagai platform hanya dengan satu kali pengerjaan. 2. Ekspansi ke Software Desktop & Web Google telah melakukan ekspansi dengan memperbarui kemampuan Flutter untuk membuat aplikasi web yang modern dan Desktop Software (Windows,macOS, dan Linux) 3. Back-End Development Support Dart juga memiliki beberapa framework yang mendukung Back-End Development, seperti Dart Frog atau Shelf dan juga mendukung pembuatan API menggunakan bahasa yang sama persis untuk membuat UI interface.
Pada saat ini, Dart menjadi salah satu bahasa pemrograman yang adaptif di era modern. Tujuan awal penciptaan Dart adalah sebagai alternatif pembuatan web, dan saat ini telah berevolusi menjadi salah satu bahasa pemrograman yang dipakai dalam Flutter, framework UI cross platform. Kelebihan yang dimiliki Dart (Null Safety,Hot Reload) dan fleksibilitasnya yang dapat dijalankan dari web, Desktop bahkan Back-End, menjadikan Dart sebagai salah satu investasi jangka panjang terbaik yang pernah dibuat Google.
